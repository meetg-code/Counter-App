# 🧮 Counter App

A clean and responsive **Counter App** built with **React.js** to strengthen my understanding of **React Hooks** and state management.

> 🚀 This project was created purely for learning and practice purposes.

---

## ✨ Features

- ➕ Increase Counter
- ➖ Decrease Counter
- 🔄 Reset Counter
- ⚡ Custom Step Value
- 📱 Responsive Design
- 🎨 Modern UI with Gradient Buttons

---

## 🛠️ Built With

- ⚛️ React.js
- 🎣 useState Hook
- 🎨 CSS3
- ⚡ Vite

---

## 🎯 What I Learned

- 📌 React Components
- 📌 useState Hook
- 📌 Event Handling
- 📌 Controlled Inputs
- 📌 Conditional Logic
- 📌 Responsive UI Design

---

## 🚀 Getting Started

```bash
git clone <repository-url>

cd counter-app

npm install

npm run dev
```
